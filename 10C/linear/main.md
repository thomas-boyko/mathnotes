## 5.1 Slope

$$m = \frac{y_2 - y_1}{x_2 - x_1}$$

| Slope | Description |
|---|---|
| $m > 0$ | Rising |
| $m < 0$ | Falling |
| $m = 0$ | Horizontal |
| undefined | Vertical |

## 5.2 Slope-Intercept Form

$$y = mx + b$$

## 5.3 General Form

$$Ax + By + C = 0$$

Point-slope: $y - y_1 = m(x - x_1)$

## 5.4 Parallel and Perpendicular

| Relationship | Condition |
|---|---|
| Parallel | $m_1 = m_2$ |
| Perpendicular | $m_1 \cdot m_2 = -1$ |
