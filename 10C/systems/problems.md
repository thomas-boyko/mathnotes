## Solving Systems of Linear Equations
All of the following problems can be solved with each of the methods we have learned: Graphing, Substitution, and Elimination. Practice identifying the easiest solution, or test each method out until you feel comfortable with it.

## Problem 1
Solve the following system of equations:

$$y = 2x - 1$$

$$3x + y = 9$$

<details>
<summary>Solution</summary>

We notice that $y$ has already been isolated in the first equation, so we will substitute it in to the second:

\[
\begin{aligned}
3x + y &= 9\\
3x+(2x-1)&=9\\
5x-1&=9\\
+1&\quad+1\\
5x&=10\\
\div 5&\quad\div5\\
x&=2
.\end{aligned}
\] 
So we have solved $x=2$! We can substitute this back into our first equation, and we get:

\[
\begin{aligned}
    y&=2x-1\\
    y&= 2(2)-1 \\
    y&= 4-1 \\
    y&= 3
\end{aligned}
\] 
So the system of equations has a solution at $x=2,$ $y=3$
</details>

## Problem 2
Solve the following system of equations:

$$2x + 3y = 12$$

$$4x - 3y = 6$$

<details>
<summary>Solution</summary>

The $y$-terms are already opposite in sign with equal coefficients, so we can eliminate directly by adding the equations:

\[
\begin{aligned}
(2x+3y)+(4x-3y) &= 12+6\\
6x &= 18\\
x &= 3
\end{aligned}
\]

Now we can substitute $x=3$ into the first equation:

\[
\begin{aligned}
2(3)+3y &= 12\\
6+3y &= 12\\
3y &= 6\\
y &= 2
\end{aligned}
\]

So we have a solution at $x=3,\ y=2$.

</details>

## Problem 3
Solve the following system of equations:

$$5x + 2y = 20$$

$$3x - 4y = -1$$

<details>
<summary>Solution</summary>

Multiply the first equation by $2$ to make the $y$-coefficients opposites:

\[
\begin{aligned}
2(5x+2y) &= 2(20)\\
10x+4y &= 40
\end{aligned}
\]

Now add to the second equation to eliminate $y$:

\[
\begin{aligned}
(10x+4y)+(3x-4y) &= 40+(-1)\\
13x &= 39\\
x &= 3
\end{aligned}
\]

Substituting $x=3$ into the first equation:

\[
\begin{aligned}
5(3)+2y &= 20\\
15+2y &= 20\\
2y &= 5\\
y &= \frac{5}{2}
\end{aligned}
\]

The solution is $x=3,\ y=\dfrac{5}{2}$.

</details>

## Problem 4
Two numbers have a sum of $48$. Twice the larger number minus the smaller number is $36$. Find both numbers.

<details>
<summary>Solution</summary>

Let the larger number be $x$ and the smaller be $y$. Set up the system:

$$x + y = 48$$

$$2x - y = 36$$

Add the equations to eliminate $y$:

\[
\begin{aligned}
(x+y)+(2x-y) &= 48+36\\
3x &= 84\\
x &= 28
\end{aligned}
\]

Substituting into the first equation:

\[
\begin{aligned}
28+y &= 48\\
y &= 20
\end{aligned}
\]

The two numbers are $28$ and $20$.

</details>

## Problem 5
A school sells adult tickets for $\$8$ and student tickets for $\$5$. They sold $200$ tickets total and collected $\$1{,}150$. How many of each ticket type were sold?

<details>
<summary>Solution</summary>

Let $a$ = adult tickets and $s$ = student tickets:

$$a + s = 200$$

$$8a + 5s = 1150$$

Multiply the first equation by $5$ and subtract:

\[
\begin{aligned}
8a+5s &= 1150\\
-(5a+5s &= 1000)\\
\hline
3a &= 150\\
a &= 50
\end{aligned}
\]

Then $s = 200 - 50 = 150$.

The school sold $50$ adult tickets and $150$ student tickets.

</details>

## Problem 6
A boat travels $36\text{ km}$ upstream in $3$ hours and $48\text{ km}$ downstream in $2$ hours. Find the speed of the boat in still water and the speed of the current.

<details>
<summary>Solution</summary>

Let $b$ = speed of boat in still water, $c$ = speed of current. Then:

$$b - c = \frac{36}{3} = 12$$

$$b + c = \frac{48}{2} = 24$$

Add the equations:

\[
\begin{aligned}
2b &= 36\\
b &= 18
\end{aligned}
\]

Substituting back:

\[
\begin{aligned}
18 - c &= 12\\
c &= 6
\end{aligned}
\]

The boat's speed in still water is $18\text{ km/h}$ and the current is $6\text{ km/h}$.

</details>

## Problem 8
How many solutions does the following system have? Explain your reasoning.

$$2x - 4y = 8$$

$$x - 2y = 4$$

<details>
<summary>Solution</summary>

Divide the first equation by $2$:

$$x - 2y = 4$$

This is identical to the second equation. The two equations represent the **same line**, so the system has **infinitely many solutions** — any point on the line $x - 2y = 4$ is a solution.

</details>

## Problem 9
How many solutions does the following system have? Explain your reasoning.

$$3x - 6y = 12$$

$$x - 2y = 7$$

<details>
<summary>Solution</summary>

Divide the first equation by $3$:

$$x - 2y = 4$$

The second equation is $x - 2y = 7$. Both lines have the same slope but different intercepts — they are **parallel and never intersect**, so the system has **no solution**.

</details>
